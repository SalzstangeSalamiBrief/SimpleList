# SimpleList
A simple list for everyone.
This Application is builded for widescreens (1920x1080).
## dev
```
// setup db
$ npm run database:setup

// add util files to dist
$ npm run copyFrontendUtil

// start server for development
$ npm run dev
```

## Production
```
// prepare dist
$ npm run init

// run app
$ npm run start
```
